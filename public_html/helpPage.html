ChowChow Help

This program is designed to build a simple online forum for students from University of Arizona. The functions from this website includes:
l Register Account, Login, Logout.
l Post threads, comment to threads.
l Add friend, delete friend.
l Update profile.

Register page

On this page, The user can register an account. The user will be asked to confirm their passwords. When the user creates account successful, the user will be take to the homepage and automatically signed in.

HomePage

This is the homepage of the website, users can post threads, view threads on this page. There has nav let users go to other pages. When the user click the title of the page, the link will go to the detail page of the post.

Login

The login page prompt users to login. If the user want to comment a post, to post thread or want to goto the personPage, they need to login first.

PersonPage

The user can view their own posts here, the headshot and username shown in this page. Users can update their information by clicking the navigation button.

DetailPage

Users can see details of the thread. Who post it, they also can leave their comments on this page.
